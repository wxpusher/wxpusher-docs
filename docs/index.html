WxPusher 消息推送 API 与接入指南

由 WxPusher 维护 · 最后更新：2026-07-11

WxPusher 是永久免费的实时消息推送服务。开发者可通过 HTTP API 把服务器告警、定时任务结果、CI/CD 状态和业务通知发送到 Android、iOS、HarmonyOS、macOS、Windows、Linux 或微信 ClawBot。

选择接入方式

  • 标准推送 API：使用 appToken，通过 UID 单发或 Topic 群发；适合业务系统，支持用户管理、回调和状态查询。
  • 极简推送 SPT：无需创建应用，扫码获取 SPT 后即可发送；适合个人脚本给自己发送通知。

标准推送快速示例

curl -X POST https://wxpusher.zjiecode.com/api/send/message \
  -H 'Content-Type: application/json' \
  -d '{"appToken":"AT_xxx","content":"服务器告警","contentType":1,"uids":["UID_xxx"]}'
